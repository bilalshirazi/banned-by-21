# Test Data Collection

Store your generated test images here. To keep the project organized, use the following naming convention:

- `hijab_[nn].png`
- `kippah_[nn].png`
- `turban_[nn].png`
- `niqab_[nn].png`
- `medical_mask_[nn].png`
- `baseline_[nn].png` (No symbols)

These images are used for calibrating and verifying the CLIP detection logic in `app.py`.
